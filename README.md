# bloguide
코딩잔향: 블로가이드 (bloguie) 프로젝트는 네이버 블로그 게시글을 마크다운으로 추출하는 내용을 포함해 마크다운 텍스트를 분석하는 분석기 API 프로젝트입니다.

## 개발 환경
- Python 3.10.6
- FastAPI (ver. 0.110.2)
- VirutalBox 6.1
- vagrant
- docker-compose

## 주요 기능
1. 네이버 게시글 -> 마크다운 포팅
2. ES 분석기 마크다운 블로그 게시글 적용
3. LLM 기반 게시글 분석
5. 이메일 알림
6. Third Party tools
7. API 관리 기능
